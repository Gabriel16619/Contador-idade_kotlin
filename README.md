 Contador de idade utilizando o Android.Studio

 - Nesta atividade desenvolvemos uma interface dinaminca onde os botões alteravem a idade (0 a 17 menor, 18 a 180 maior), o botão com o sinal de "-" diminui -1 numero, enquanto o botão com o "+" aumenta uma idade aumenta +1 numero
